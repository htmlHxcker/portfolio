Privacy Policy

Last Updated: 24/10/2024

What We Do

Site Colour Palette Extractor is a browser extension that analyzes the currently active webpage to extract color information. We only access the current webpage to identify and extract color data used in the site's design.

Data Collection

We collect the following information:

  • Color values from the webpage you're visiting (including background colors and text colors)

How We Use Your Data

The color data we collect is:

  • Only used to display the color palette within the extension
  • Only stored locally on your device if you choose to save palettes
  • Never transmitted to our servers
  • Never shared with third parties

Data Storage

Any palettes you save are stored locally in your browser using Chrome's storage API. You can delete this data at any time by clearing your browser data or uninstalling the extension.
